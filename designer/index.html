Select tables

Use this section to select the DwC-DP tables needed for your dataset. Start with at least one table from the first column (tables with no required dependencies). Only tables whose required dependencies are already selected can be checked. The field search above can help you identify and select which tables you need. Table selections affect relationship visualizations, table mapping order, and output generation.

Visualize relationships

This section shows how tables relate to each other. Each table is represented as a node in the graph, while each relationship is represented by an arrow between nodes (an "edge"). Click on a node or edge to see a popup showing detailed information about it. By default, only tables that have been specifically selected (either with its checkbox, or by selecting a field from that table) show in the graph. "Enabled" tables are those that have all of the other tables they depend on selected. These can be revealed by clicking on "Show enabled tables". Right-click on an enabled table to select that table (and potentially enable more tables). Right-click on a "Selected" table to deselect it and any fields that might have been selected within it.
Select or enable tables to populate the graph.

Recommended table mapping order

Use this section after selecting tables. It generates a recommended order for mapping data into the selected tables based on all foreign-key relationships among them. This does not change the selected tables, it only helps you plan the sequence of mapping work.

Generate outputs

Use this section after selecting tables. Generate a datapackage.json for inspection and copying, or download a bundle containing the datapackage.json plus one empty CSV file with headers for each selected table. The outputs always reflect the current table selection.
